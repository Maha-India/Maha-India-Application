# Security Policy

## Supported Versions

 **We do not recommend using the versions marked with a :x: as they do not support encryption or have an unresolved security issue.**

| Version | Supported          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |
